# Support

## Getting Help

Use issues for:
1. Bug reports
2. Language/runtime behavior questions
3. Tooling problems (`nypm`, `nyfmt`, `nydbg`)

Include:
1. OS and shell
2. compiler (`clang/gcc/cl`) and version
3. exact command run
4. full error output
5. minimal `.ny` reproduction

## Response Expectations

1. Best effort support on `main`.
2. Production incidents should include severity and business impact.
3. Security-sensitive reports should follow `SECURITY.md`.
